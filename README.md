# 🔗 Projeto - Simulação de Linktree

Uma página de links pessoal inspirada no Linktree, desenvolvida com HTML e CSS puro. Exibe foto de perfil, nome, descrição e botões de redes sociais de forma responsiva e estilizada.

## ✨ Funcionalidades

- 🖼️ Foto de perfil centralizada com estilo circular
- 👤 Nome e descrição do usuário
- 🔗 Botões de redes sociais com links clicáveis
- 📱 Layout responsivo para mobile e desktop

---

## 🛠️ Tecnologias Utilizadas

- **HTML5** - estrutura semântica da página
- **CSS3** - estilização, layout e responsividade

---

## 📁 Estrutura de Pastas

```
projeto-linktree/
│
└── src/
    ├── assets/
    │   ├── icons/
    │   │   └── favicon.ico       # Ícone da aba do navegador
    │   └── img/
    │       └── foto-linkedin.png # Foto de perfil
    ├── css/
    │   └── style.css             # Estilos e layout
    └── index.html                # Estrutura principal da página
```

---

## 🚀 Como Executar

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/GiovannaParejas/projeto-linktree
   ```

2. **Acesse a pasta do projeto:**
   ```bash
   cd projeto-linktree
   ```

3. **Abra o arquivo `index.html`** no navegador de sua preferência.

---
 
## ✏️ Como Personalizar
 
Para adaptar a página com suas próprias informações, edite os arquivos dentro de `src/`:
 
| Campo | Arquivo | O que alterar |
|---|---|---|
| Foto de perfil | `assets/img/foto-linkedin.png` | Substitua pela sua foto |
| Ícone da aba | `assets/icons/favicon.ico` | Substitua pelo seu favicon |
| Nome | `index.html` | Altere o texto com seu nome |
| Descrição | `index.html` | Altere o texto da bio |
| Links sociais | `index.html` | Atualize os `href` dos botões com suas URLs |
| Cores e fontes | `css/style.css` | Personalize o visual da página |
